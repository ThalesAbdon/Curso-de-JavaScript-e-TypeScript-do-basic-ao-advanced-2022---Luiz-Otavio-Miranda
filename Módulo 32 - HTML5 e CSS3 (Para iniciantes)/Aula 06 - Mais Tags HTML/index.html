O elemento HTML cite representa uma referência a um trabalho artístico. Deve incluir o título do trabalho ou uma URL de referência, que pode ser em uma forma abreviada de acordo com as convenções usadas para a adição dos metadados de citação.

O elemento de dados HTML vincula um determinado conteúdo a uma tradução legível por máquina. Se o conteúdo estiver relacionado à hora ou à data, a hora deve ser usada.

  • Mini Ketchup
  • Jumbo Ketchup
  • Mega Jumbo Ketchup

O elemento HTML Variable <var> representa uma variável em uma expressão matemática ou um contexto de programação.: x = y + 2

O Elemento HTML <caption> (ou Elemento HTML Subtitulo de Tabela) representa o título de uma tabela. Embora ele seja sempre o primeiro filho de um <table>, seu estilo, usando CSS pode colocar ele em qualquer lugar relativo a tabela.
